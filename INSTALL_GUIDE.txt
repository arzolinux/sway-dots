dotfiles install for sway arch

this guide is for installing Buage Dots without the pain of 5000 errors. 

## Prerequisites
- arch linux 
- sway installed: `pacman -S sway wayland xorg-server-xwayland`
- good internet so mirrors dont fuck it up 

## Installation Steps

1. **clone the dottys**
   ```
   git clone https://github.com/arzolinux/sway-dots.git
   cd ~/.dotfiles
   ```

2. **make the install script like an exe**
   ```
   chmod +x install.sh
   ```

3. **Run the installation script**
   ```
   ./install.sh
   ```
   The script will:
   - ask u for ur password because it can
   - install everything needed
   - create symlinks
   - copy wallpapers to pictures folder
   - install tmux pluggys

4. **Apply the changes**
   - logout
   - login again
   - shutdown/restart whole pc

## Customization
- edit the config in the files on ur fucking pc if u dont know how to get here u dont deserve using these dots
- The symlinks will automatically reflect changes
- for timmy tuff knuckles tmux plugins, add/remove from tmux.conf and run `~/.tmux/plugins/tpm/bin/install_plugins`

## Troubleshooting
- run `pacman -Syu` make sure everythings up to date
- check that everything installed
- refer to sway documentation for sway issues
